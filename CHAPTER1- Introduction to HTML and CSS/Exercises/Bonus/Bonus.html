HTML & CSS Quick Reference Guide

HTML Tags

<h1> - Heading Tag

Defines a main heading on the page.

<h1>Welcome!</h1>

Welcome!

<p> - Paragraph Tag

Used to define a block of text.

<p>This is a paragraph.</p>

This is a paragraph.

<b> - Bold

Makes the text bold.

<b>Bold Text</b>
Bold Text

<br> - Line Break

Creates a single line break.

Line one.<br>Line two.
Line one.
Line two.

<i> - Italic

Makes the text italic.

<i>Italic Text</i>
Italic Text

<header> - Header Section

Defines a page header or introductory content.

<header>Welcome Section</header>
Welcome Section

<img> - Image

Embeds an image into the page.

<img src="hhttps://images.unsplash.com/photo-1743482858217-5aef42cfc636?q=80&w=1770&auto=format&fit=crop&ixlib=rb-4.0.3&ixid=M3wxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8fA%3D%3D" alt="placeholder">

<ul>, <ol>, <li> - Lists

<ul> creates an unordered list, <ol> an ordered list, and <li> each list item.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
  </ul>
  
  <ol>
    <li>First</li>
    <li>Second</li>
  </ol>
  • Item 1
  • Item 2
  1. First
  2. Second

CSS Properties

color

Sets the color of text.

p {
  color: red;
}

This text is red.

background-color

Changes the background color of an element.

div {
  background-color: lightgray;
}
Background color applied.

background-image

Sets a background image on an element.

div {
  background-image: url('https://images.unsplash.com/photo-1743482858217-5aef42cfc636?q=80&w=1770&auto=format&fit=crop&ixlib=rb-4.0.3&ixid=M3wxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8fA%3D%3D');
}

width

  width: 200px;
}
Width set to 200px

font-family

Sets the font style of text.

p {
  font-family: 'Courier New', monospace;
}
Courier New font

height

Sets the height of an element.

div {
  height: 100px;
}
Height set to 100px
